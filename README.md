Desarrollo de una Single Page Application (SPA) para un e-commerce completo (tienda de neumáticos), enfocada en una experiencia de usuario fluida y segura. La aplicación gestiona todo el ciclo de vida del cliente: desde el registro y autenticación, pasando por el manejo del carrito de compras, hasta el procesamiento del checkout y la visualización detallada del estado de las órdenes. El proyecto destaca por su arquitectura limpia, separando estrictamente la lógica de negocio, el estado global y la interfaz de usuario.
