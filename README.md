Desarrollo de una API REST escalable para un ecommerce especializado en la venta de llantas. La plataforma permite autenticación con credenciales y login social, gestión de usuarios, catálogo de productos, procesamiento de órdenes y envío automático de confirmaciones por correo. La arquitectura sigue un patrón modular basado en controllers, services y middlewares. El sistema fue implementado con Node.js, Express y PostgreSQL, utilizando JSON Web Token para autenticación segura e integrado con Resend para notificaciones de órdenes.
